# 加入場景物件(AI 無法走過)

1. Inspector > ☑ Static
2. Window > AI > Navigation > Object > Walkable
3. Navigation > Bake > Bake  
   <br/>

## 會移動的物件(AI 無法走過)

1. Add Component > Nav Mesh Obstacle > ☑ Carve
2. Window > AI > Navigation > Object > Walkable  
   <br/>
